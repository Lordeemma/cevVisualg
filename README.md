# cevVisualg
Repositório feito para arquivar algoritmos feitos por mim no VisualG, seguindo as instruções contidas nas aulas e no PDF de exercícios do Curso em Vídeo. Alguns algoritmos podem conter comandos anacrônicos às aulas, pois acabo pesquisando formas de realizar certas operações. Use este repositório como possível fonte de consulta para os problemas propostos pelo professor Gustavo Guanabara.

Os algoritmos estão separados por origem (aula, PDF) e por conteúdo (condições, loops...). Desafios sempre possuem o sufixo "D" ao final do nome do arquivo, além de serem armazenados em sua própria pasta, mas não são categorizados por conteúdo. Algoritmos encontrados separadamente foram criados apenas para fins de teste de conhecimento e podem não possuir relação com os exercícios propostos.
